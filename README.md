# 🚀 The Wild Oasis Dashboard 

## 🔍 Overview
A powerful dashboard that allows users to add, edit, and delete data with ease. It also provides insightful analytics and interactive diagrams to help visualize and understand key information efficiently.

---

## 🛠️ Technologies Used


- **React.js** as the React framework
- **Styled CSS** for styling tags
- **React Toastify** for notifications 
- **recharts**  libraries for making a nice diagrams to help visualize and understand key 
- **React Router** for navigate between pagese
---

## ✨ Features

- 📄 View all job Cabins
- 🔍 View job Cabins page
- ➕ Add new Cabin
- ✏️ Edit existing Cabin
- 🗑️ Delete Cabins
- ⚡ Lazy loading using React.lazy and Suspense
- 🎨 Styled CSS for styling tags 

- 🚀 Performance optimized (CLS, bundle size, code splitting)
---

### To Install and Run the Project
1. Clone the repository:
    ```bash
   git clone https://github.com/Lotfy12/the-wild-oasis-dashboard.git
    ```
2. Navigate to the project directory:
    ```bash
    cd project-directory
    ```
3. Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
4. Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
